RIFURN – Responsive Furniture Website

RIFURN is a responsive furniture website built using semantic HTML, CSS, and JavaScript.
The project focuses on clean layout structure, mobile-first responsiveness, and basic UI interactions without using external frameworks.

This project was created to strengthen frontend fundamentals and understand how real websites adapt across different screen sizes.

Features

Responsive navigation bar that adapts to different screen sizes

Furniture product section with structured card layout

Image-based UI optimized for multiple screen resolutions

Mobile-first design using CSS media queries

Reusable CSS components for consistent styling

Smooth hover effects and basic UI interactions using JavaScript

Tech Stack

HTML5 (Semantic Markup)

CSS3 (Flexbox, Media Queries)

JavaScript (Vanilla JS)

How to Run Locally

Clone the repository

Navigate into the project folder
cd responsive_furniture_website

Open index.html in your browser

No additional dependencies are required.
